Важно

Восстановить пароль невозможно. При утере пароля от сейфа, сейф придется отключить из приложения. Восстановить сейф можно только из цифровой копии ( brainwallet) или физической копии файла с другим паролем, который вы помните.

Сохранить файл сейфа
Чтобы не потерять и надежно защитить файл сейфа, вам необходимо создать бекап файла сейфа на следующем шаге
Продолжить
Файл сейфа является хранищем ваших средств. История платежей и Контакты также хранится в этом файле. Чтобы файл сейфа оставался подключенным в приложение, не перемещайте его из той директории, куда вы его сохранили.

Физический бекап файла сейфа

Скопируйте созданный файл сейфа в другое надежное место, не храните бекап на том же компьютере.
